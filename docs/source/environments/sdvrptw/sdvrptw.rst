========
SDVRPTW
========

Split Delivery Vehicle Routing Problem with Time Windows (SDVRPTW)

The Split Delivery Vehicle Routing Problem with Time Windows (SDVRPTW) is a generalization of the CVRPTW where each customer can be visited more than once by several vehicles and a fraction of the demand can be met (see [Bia19]_).
